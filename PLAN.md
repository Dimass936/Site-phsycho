# План разработки: Сайт психолога

Подробный план создания многостраничного статического сайта психолога на основе спецификации и архитектуры.

---

## Этап 1: Подготовка и настройка проекта

**Что делается:**
- Создание структуры директорий проекта (`site/`, `site/css/`, `site/css/pages/`, `site/js/`, `site/img/`)
- Инициализация Git-репозитория
- Создание базовых файлов: `index.html`, `css/base.css`, `css/components.css`, `css/layout.css`
- Настройка подключения Google Fonts (Inter + Playfair Display)
- Создание `ARCHITECTURE.md` и `SPECIFICATION.md`

**Задействованные агенты:**

| Агент | Скилы | Задача |
|-------|-------|--------|
| **Frontend Developer** | HTML5, CSS3, Git | Структура проекта, базовые файлы, подключение шрифтов |
| **UI/UX Designer** | Дизайн-системы, цветовая теория | Определение палитры, типографики, spacing-системы |

**Самостоятельно ИИ:** ✅ Вся техническая настройка, создание директорий, базовых файлов, дизайн-системы.

**Нужно от пользователя:**
- Название/имя для логотипа сайта (сейчас заглушка «Имя Фамилия»)
- Подтверждение цветовой палитры (или правки)

---

## Этап 2: Создание общих компонентов CSS

**Что делается:**
- Написание стилей для кнопок (primary, secondary, warm, large, full)
- Стили карточек (card, card--flat, service-card)
- Стили секций (section, section--bg, section__header)
- Стили мессенджеров (messenger-link с брендовыми цветами)
- Стили форм (form__input, form__textarea, form__select, form__error)
- Бейджи, разделители, утилиты
- Лейаут: хедер (fixed, logo, nav, burger), футер (4 колонки), сетка (grid--2/3/4), flex-утилиты
- Адаптивность: медиа-запросы для tablet (≤1023px), mobile (≤767px), small mobile (≤479px)

**Задействованные агенты:**

| Агент | Скилы | Задача |
|-------|-------|--------|
| **Frontend Developer** | CSS3, custom properties, flexbox, grid, медиа-запросы | Все CSS-компоненты, лейаут, адаптив |
| **UI/UX Designer** | Микровзаимодействия, hover-эффекты, accessibility | Анимации, фокус-состояния, контраст |

**Самостоятельно ИИ:** ✅ Все CSS-стили, адаптивность, hover-эффекты.

**Нужно от пользователя:** ❌ Не требуется на этом этапе.

---

## Этап 3: JavaScript — интерактивность

**Что делается:**
- `main.js`: мобильное меню (бургер, открытие/закрытие, блокировка скролла), активные ссылки, плавная прокрутка к якорям, тень хедера при скролле
- `faq.js`: аккордеон FAQ (раскрытие одного закрывает остальные, анимация max-height)
- `form.js`: валидация формы (имя, способ связи, контакт), сообщения об ошибках, сообщение об успехе

**Задействованные агенты:**

| Агент | Скилы | Задача |
|-------|-------|--------|
| **JavaScript Developer** | Vanilla JS ES6+, IIFE, event delegation, DOM, валидация форм | Все JS-модули |

**Самостоятельно ИИ:** ✅ Вся логика интерактивности.

**Нужно от пользователя:** ❌ Не требуется на этом этапе.

---

## Этап 4: Главная страница (index.html)

**Что делается:**
- Хедер с навигацией и мобильным меню
- Hero-секция: бейдж, заголовок, описание, CTA-кнопки, кнопки мессенджеров, заглушка для фото
- Секция направлений: 6 карточек услуг (детская психология, отношения, стресс, подростки, личностный рост, кризисы)
- Секция «Обо мне» превью: текст, 3 карточки преимуществ
- Секция «Как проходит» превью: 4 шага
- Секция отзывов: 3 карточки с цитатами, аватарами, именами
- CTA-баннер (зелёный фон)
- Футер с 4 колонками

**Задействованные агенты:**

| Агент | Скилы | Задача |
|-------|-------|--------|
| **Frontend Developer** | HTML5, семантическая разметка, BEM | Вёрстка главной страницы |
| **Content Writer** | SEO-копирайтинг, эмпатичный стиль | Тексты hero, направлений, отзывов, CTA |
| **UI/UX Designer** | Композиция, визуальная иерархия | Расположение элементов, баланс секций |

**Самостоятельно ИИ:** ✅ Вёрстка, тексты-заглушки, структура, SEO-теги.

**Нужно от пользователя:**
- Ваше фото (или разрешение использовать заглушку)
- Реальные отзывы клиентов (сейчас — примеры)
- Реальный текст «Обо мне» (количество лет опыта, подходы)

## Этап 11: SEO-оптимизация

**Что делается:**
- Проверка уникальных `<title>` и `<meta description>` для каждой страницы
- Добавление Open Graph тегов (`og:title`, `og:description`, `og:image`, `og:type`)
- Проверка структуры заголовков (один H1 на страницу, правильная иерархия H2-H4)
- Добавление Schema.org разметки (LocalBusiness, Person, FAQPage)
- Проверка семантической разметки (header, main, section, article, footer)
- Добавление `hreflang="ru"` для русскоязычного контента
- Canonical URLs

**Задействованные агенты:**

| Агент | Скилы | Задача |
|-------|-------|--------|
| **SEO Specialist** | Техническое SEO, Schema.org, Open Graph | Все мета-теги, разметка, аудит |
| **Frontend Developer** | HTML5, семантика | Реализация рекомендаций |

**Самостоятельно ИИ:** ✅ Мета-теги, структура заголовков, Schema.org, семантика.

**Нужно от пользователя:**
- Ключевые слова для продвижения (или ИИ подберёт)
- OG-изображение для соцсетей (или заглушка)

## Минимальный вход от пользователя для запуска MVP

Для быстрого запуска достаточно предоставить **4 вещи**:

1. **Имя** (для логотипа и копирайта)
2. **Фото** (для главной и страницы «Обо мне»)

Всё остальное ИИ-агенты создадут самостоятельно на основе спецификации.
