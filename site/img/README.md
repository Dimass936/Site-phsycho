# Изображения сайта психолога

## Структура

```
img/
├── favicon.svg              # Фавиконка (32×32, буква «П» на зелёном фоне)
├── og-image.svg             # Open Graph изображение (1200×630)
├── blog-placeholder.svg     # Заглушка для обложек статей блога (800×400)
├── photo-placeholder.svg    # Заглушка для фото психолога (200×200, силуэт)
├── icons/
│   ├── telegram.svg         # Иконка Telegram (24×24)
│   ├── whatsapp.svg         # Иконка WhatsApp (24×24)
│   ├── instagram.svg        # Иконка Instagram (24×24)
│   ├── vk.svg               # Иконка VK (24×24)
│   └── max.svg              # Заглушка (24×24, круг)
└── README.md
```

## Что нужно заменить на реальные изображения

### 1. Фото психолога
- **Файл:** Замените `photo-placeholder.svg` в `index.html` (hero-секция)
- **Рекомендуемый формат:** WebP или JPEG
- **Рекомендуемый размер:** 600×600 px (квадрат) или 400×500 px (портрет)
- **Качество:** 80–85% для JPEG, 80% для WebP
- **Фон:** Однотонный или размытый, тёплые тона
- **Совет:** Добавьте `loading="eager"` и `width`/`height` атрибуты

### 2. Фавиконка
- **Текущий файл:** `favicon.svg` (векторная буква «П»)
- **Для полной поддержки** рекомендуется дополнительно создать:
  - `favicon-32.png` — 32×32 px
  - `favicon-16.png` — 16×16 px
  - `apple-touch-icon.png` — 180×180 px
  - `android-chrome-192.png` — 192×192 px
  - `android-chrome-512.png` — 512×512 px
- **Инструмент:** [realfavicongenerator.net](https://realfavicongenerator.net/)

### 3. Open Graph изображение
- **Текущий файл:** `og-image.svg` (1200×630)
- **Для совместимости** рекомендуется также создать `og-image.png` (1200×630)
- Некоторые платформы (VK, некоторые мессенджеры) не поддерживают SVG для OG

### 4. Обложки статей блога
- **Текущий файл:** `blog-placeholder.svg` (800×400)
- **Рекомендуемый формат:** WebP
- **Рекомендуемый размер:** 800×400 px (соотношение 2:1)
- Создайте отдельные файлы для каждой статьи

### 5. Иконки мессенджеров
- **Статус:** Готовы к использованию (SVG)
- **Цвет:** Наследуется через `currentColor` — контролируется CSS
- Если нужны цветные версии, замените на официальные логотипы

## Рекомендации по оптимизации

1. **Форматы:** WebP > JPEG > PNG для фотографий, SVG для иконок и графики
2. **Размеры:** Не загружайте изображения больше, чем нужно для отображения
3. **Lazy loading:** Добавьте `loading="lazy"` для изображений ниже первого экрана
4. **Alt-текст:** Всегда указывайте осмысленный `alt` для доступности
5. **CDN:** Рассмотрите использование CDN для раздачи изображений
