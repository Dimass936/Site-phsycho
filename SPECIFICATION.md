# Спецификация проекта: Сайт психолога

## 1. Общее описание

Многостраничный статический сайт психолога для онлайн-консультаций.
Целевая аудитория: взрослые, пары, подростки.
Специализации: детская психология, отношения, стресс, тревожность.

## 2. Цели сайта

- Привлечение новых клиентов через онлайн-консультации
- Информирование о методах работы и подходах
- Создание доверия через профессиональный контент
- Упрощение записи на консультацию через мессенджеры

## 3. Структура страниц

| Страница | Файл | Описание |
|----------|------|----------|
| Главная | `index.html` | Hero-секция, направления работы, отзывы, призыв к действию |
| Обо мне | `about.html` | Образование (таймлайн), подход, философия, сертификаты |
| Услуги | `services.html` | 6 направлений с описанием, ценами, кнопками записи |
| Как проходит консультация | `how-it-works.html` | 5 шагов процесса, преимущества онлайн-формата, FAQ |
| Блог | `blog.html` | Карточки статей с фильтрацией по категориям |
| FAQ | `faq.html` | 8 вопросов-ответов в формате аккордеона |
| Контакты | `contact.html` | Мессенджеры, соцсети, форма обратной связи |

### 4.2 Типографика

- **Основной шрифт**: Inter / system-ui
- **Заголовки**: Playfair Display (serif)
- **Размеры**:
  - H1: 2.25rem (36px) / 3rem (48px) для hero
  - H2: 1.875rem (30px)
  - H3: 1.5rem (24px)
  - H4: 1.25rem (20px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)
  - XS: 0.75rem (12px)

### 4.3 Отступы

- Базовая единица: 8px
- 2xs: 4px | xs: 8px | sm: 12px | md: 16px | lg: 24px | xl: 32px | 2xl: 48px | 3xl: 64px | 4xl: 96px

### 4.4 Компоненты

- **Кнопки**: скругление 8px, плавные переходы 250ms
- **Карточки**: скругление 12px, мягкая тень, hover с подъёмом
- **Формы**: скругление 6px, фокус с обводкой зелёным
- **Бейджи**: скругление полное (pill), цветные фоны
- **Мессенджеры**: pill-кнопки с брендовыми цветами

## 5. Функциональные требования

### 5.1 Навигация

- Фиксированный хедер с логотипом и меню
- Адаптивное мобильное меню (бургер)
- Активная ссылка подсвечивается зелёной линией
- Футер с 4 колонками: бренд, навигация, информация, соцсети

### 5.2 Интерактивность

- Мобильное меню: открытие/закрытие бургером, блокировка скролла
- Плавная прокрутка к якорям
- Аккордеон FAQ: раскрытие одного вопроса закрывает остальные
- Валидация формы: проверка полей, сообщения об ошибках, успех
- Фильтрация блога: кнопки категорий, показ/скрытие карточек
- Хедер: тень при скролле

### 5.3 Интеграции

- Кнопки мессенджеров: Telegram, WhatsApp
- Ссылки на соцсети: Instagram, VK, Max
- Все ссылки открываются в новой вкладке с `rel="noopener"`

### 5.4 Адаптивность

| Брейкпоинт | Ширина | Описание |
|------------|--------|----------|
| Desktop | 1024px+ | Полная сетка, десктоп-меню |
| Tablet | 768px – 1023px | 2 колонки, бургер-меню |
| Mobile | < 768px | 1 колонка, уменьшенная типографика |
| Small mobile | < 480px | Минимальные отступы, компактные элементы |

### 5.5 Производительность

- Без внешних фреймворков (чистый HTML/CSS/JS)
- Шрифты Google Fonts с `preconnect`
- Минимизация запросов
- Быстрая загрузка (< 2 сек)

## 6. SEO требования

- Семантическая разметка (`header`, `main`, `section`, `article`, `footer`)
- Уникальные `<title>` и `<meta description>` для каждой страницы
- Open Graph теги (`og:title`, `og:description`, `og:type`)
- Структура заголовков: один H1, H2 для секций, H3/H4 для подсекций
- Адаптивность для мобильных
- `hreflang="ru"` для русскоязычного контента

## 7. Технические требования

### 7.1 Стек

- **HTML5** — семантическая разметка
- **CSS3** — custom properties, flexbox, grid, медиа-запросы
- **Vanilla JavaScript** — IIFE-паттерн, event delegation
- **Google Fonts** — Inter + Playfair Display

### 7.2 Архитектура CSS

- **base.css** — сброс, переменные, утилиты
- **components.css** — кнопки, карточки, формы, бейджи, разделители
- **layout.css** — хедер, футер, сетка, флекс-утилиты
- **responsive.css** — медиа-запросы для всех брейкпоинтов
- **pages/*.css** — стили для конкретных страниц

### 7.3 Архитектура JS

- **main.js** — мобильное меню, активные ссылки, плавная прокрутка, тень хедера
- **faq.js** — аккордеон FAQ
- **form.js** — валидация формы записи

## 8. Контент

- Профессиональный, но тёплый тон
- Акцент на доверии и безопасности
- Простой язык без излишнего академизма
- Призывы к действию на каждой странице
- Тексты-заглушки с пометками для замены

# ИИ-Агенты для разработки проекта

## 1. Frontend Developer Agent

**Назначение**: Вёрстка HTML/CSS, создание страниц и компонентов

**Необходимые скилы**:
- HTML5 (семантическая разметка, доступность, SEO-теги)
- CSS3 (custom properties, flexbox, grid, медиа-запросы, анимации)
- BEM-подобная методология именования классов
- Адаптивная вёрстка (mobile-first или desktop-first)
- Кроссбраузерная совместимость
- Google Fonts интеграция
- SVG-иконки inline
- Оптимизация производительности (minification, critical CSS)

## 2. JavaScript Developer Agent

**Назначение**: Интерактивность, валидация, обработка событий

**Необходимые скилы**:
- Vanilla JavaScript (ES6+)
- IIFE-паттерн для изоляции модулей
- Event delegation
- DOM-манипуляции
- Валидация форм (клиентская)
- Аккордеон-компоненты
- Фильтрация контента
- Плавная прокрутка (smooth scroll)
- Обработка скролла (тень хедера)
- Мобильное меню (бургер, блокировка скролла)

## 3. UI/UX Designer Agent

**Назначение**: Дизайн-система, цветовая палитра, типографика, компоненты

**Необходимые скилы**:
- Дизайн-системы и компонентный подход
- Цветовая теория (пастельные палитры, контраст, доступность)
- Типографика (шрифтовые пары, иерархия, читаемость)
-Spacing-системы (8px grid)
- Микровзаимодействия (hover-эффекты, переходы)
- Мобильный UX (адаптивные паттерны, тач-интерфейсы)
- Accessibility (WCAG, контраст текста, фокус-состояния)
- Психология цвета (успокаивающие тона для психолога)

## 4. Content Writer / Copywriter Agent

**Назначение**: Тексты для всех страниц, SEO-копирайтинг

**Необходимые скилы**:
- Копирайтинг для сайтов услуг
- SEO-копирайтинг (мета-теги, ключевые слова, заголовки)
- Тон голоса: профессиональный + тёплый + доверительный
- Написание текстов для психологической тематики
- Создание контента для FAQ
- Написание заголовков и CTA (call-to-action)
- Текст для страниц: Hero, Обо мне, Услуги, Блог, Контакты
- Эмпатичный стиль (учёт эмоционального состояния читателей)

## 5. SEO Specialist Agent

**Назначение**: Оптимизация для поисковых систем

**Необходимые скилы**:
- Техническое SEO (мета-теги, семантическая разметка, Open Graph)
- Структура заголовков (H1-H6)
- Schema.org разметка (LocalBusiness, Person, FAQPage)
- Оптимизация скорости загрузки
- SEO-аудит контента
- Ключевые слова для психологических услуг
- URL-структура и внутренняя перелинковка
- Canonical URLs

## 6. QA / Testing Agent

**Назначение**: Проверка корректности, кроссбраузерность, доступность

**Необходимые скилы**:
- Кроссбраузерное тестирование (Chrome, Firefox, Safari, Edge)
- Тестирование адаптивности (эмуляция устройств)
- Accessibility-аудит (Lighthouse, axe)
- Валидация HTML/CSS (W3C validator)
- Функциональное тестирование (формы, меню, аккордеон, фильтры)
- Performance-тестирование (Lighthouse, PageSpeed Insights)
- Тестирование на реальных устройствах

## 7. DevOps / Deployment Agent

**Назначение**: Публикация сайта, хостинг, домен, CI/CD

**Необходимые скилы**:
- Статический хостинг (GitHub Pages, Netlify, Vercel, Cloudflare Pages)
- DNS и домены
- HTTPS/SSL сертификаты
- Оптимизация для CDN
- Автоматический деплой из Git
- Минификация HTML/CSS/JS
- Оптимизация изображений (WebP, lazy loading)
- Мониторинг доступности и скорости

## 8. Image/Media Processing Agent

**Назначение**: Обработка фото, создание заглушек, оптимизация изображений

**Необходимые скилы**:
- Обработка фотографий (кроп, ресайз, оптимизация)
- Форматы изображений (WebP, AVIF, JPEG, PNG)
- SVG-графика (иконки мессенджеров, декоративные элементы)
- Создание заглушек для фото
- Оптимизация размеров файлов
- Lazy loading изображений
- Responsive images (`srcset`, `sizes`)
- Open Graph изображения

## Минимальный набор агентов для MVP

Для запуска базовой версии сайта достаточно **4 агентов**:

| Агент | Что делает |
|-------|-----------|
| **Frontend Developer** | Вёрстка всех страниц, CSS, структура |
| **JavaScript Developer** | Меню, формы, аккордеон, фильтры |
| **Content Writer** | Тексты для всех 7 страниц |

Остальные агенты подключаются для улучшения качества: дизайнер — для полировки UI, SEO-специалист — для продвижения, QA — для проверки, медиа — для фото.
